Connecting to a Database

To establish a connection to a MySQL database, use the following parameters:

private final String URL = "jdbc:mysql://localhost:3306/";
private final String USER = "root";
private final String PSW = "";
private Connection conn;

Executing Queries

To execute a SQL query, use the following method:

public static void ejecutarConsulta(Connection conexion, String consulta) {
    try (Statement stmt = conexion.createStatement()) {
        stmt.executeUpdate(consulta);
    } catch (SQLException ex) {
        Logger.getLogger(Consultas.class.getName()).log(Level.SEVERE, null, ex);
    }
}

Reading Files and Creating Databases

public static void consultarFichero(Connection conexion, String file) {
    try (BufferedReader reader = new BufferedReader(new FileReader(file))) {
        String line, sentencia = "";
        while ((line = reader.readLine()) != null) {
            line = line.trim();
            if (!line.isEmpty() && !line.startsWith("--")) {
                sentencia += line + " ";
                if (line.endsWith(";")) {
                    ejecutarConsulta(conexion, sentencia.trim());
                    sentencia = "";
                }
            }
        }
    } catch (IOException ex) {
        Logger.getLogger(Consultas.class.getName()).log(Level.SEVERE, null, ex);
    }
}

Converting Objects to Bytes

To convert any object to bytes using BufferedReader, you can use:

ByteArrayOutputStream bos = new ByteArrayOutputStream();
ObjectOutputStream out = new ObjectOutputStream(bos);
out.writeObject(myObject);
out.flush();
byte[] objectBytes = bos.toByteArray();

Converting Bytes Back to Object

ByteArrayInputStream bis = new ByteArrayInputStream(objectBytes);
ObjectInputStream in = new ObjectInputStream(bis);
MyClass originalObject = (MyClass) in.readObject();
